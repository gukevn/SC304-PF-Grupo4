## Objetivo

Desarrollar una aplicación que simule el juego de naipes “The Sandwich Guy”.
En este segundo avance se implementan las estructuras de datos necesarias y una interfaz gráfica capaz de mostrar visualmente las cartas distribuidas en la Caja, el Mazo, la Mano y el Pozo.

---

## Avance logrado

Cada sección muestra las cartas de manera visual. Al presionar “Barajar (Caja > Mazo)”, las cartas se mezclan aleatoriamente y se visualizan con su símbolo y color (rojo o negro).

## Instructivo

### Instalación
1. Verificar que Java SE 21 (JDK 21) esté instalado.
2. Clonar el repositorio SC304-PF-Grupo4 desde GitHub.
3. Abrir el proyecto en Visual Studio Code.

### Compilación
1. En el IDE, abrir el archivo principal Main.
2. Ejecutar con “Run” o compilar desde la terminal usando `javac`.
